# Special Topics - Web Application Frameworks

If you're in the web development world, you'll know what Angular, React, and Vue are. These are libraries/frameworks that allow you to create reactive web applications easily. You can learn how to program with these libraries if you read the INFO 343 book to the end. Using them with the iGEM wiki has a slew of difficulties that must be overcome, but going through this procedure is necessary for any team to do unless they REALLY want to, like me.

Basically, using frameworks is SUPER optional. Unless you're a hardcore web developer, knowing the contents of this chapter is going to be completely useless.
